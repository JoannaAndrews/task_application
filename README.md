● For this project, I developed a full-stack web app which enables users to create an account and access their upcoming tasks

● Users can add new tasks, edit tasks, and delete existing tasks

● Users can also view an overall summary of pending and completed tasks by going to the 'Analytics' page 

● The application also features a 'Study Timer' feature through which provides a timer that can be used to time study sessions and task completion 

● To perform user authentication and to maintain user information, I used Firebase authentication and Firestore. I created collections for the users and for the tasks associated with different users.
